<H1>NAME</H1>
<P>
xselman - X selection management.
</P>
<H1>SYNOPSIS</H1>
<P>
**xselman**
</P>
<H1>DESCRIPTION</H1>
<P>
**xselman** is a program that manages X selections.
</P>
<P>
For now, that means that it synchronizes the primary selection with the
clipboard (and vice-versa).
</P>
<H1>OPTIONS</H1>
<P>
**-h** Help.
</P>
